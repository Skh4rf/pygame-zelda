# pygame-zelda
A WDIC project that aims to improve an existing pygame project. Original: https://github.com/clear-code-projects/Zelda

## Keyboard Layout for the Game
- **Movement:**
  - **Up:** Move forward
  - **Down:** Move backward
  - **Left:** Move left
  - **Right:** Move right

- **Actions:**
  - **Space:** Attack
  - **LCTRL:** Use spell

- **Switching:**
  - **Q:** Change weapon
  - **E:** Change spell

- **Menu:**
  - **ESC:** Game menu
  - **M:** Ingame upgrade menu
